Calculadora de expressões numéricas

Oque são expressões numéricas ?

Expressões numéricas são conteúdos que cai normalmente do sexto ao sétimo ano do Ensino Fundamental Mas você acaba usando pela sua vida toda pois depois no sétimo oitavo nono ano eu no ensino médio vai cair muito conteúdo sobre equações equações são uma expressão por isso sempre aprendemos Primeiro as expressões numéricas O que é basicamente a base de tudo que vem depois na álgebra .

As expressões numéricas são nada mais nada menos que um conjunto de duas ou mais operações agrupadas abaixo temos como você efetua o cálculo de uma expressão numérica :

como resolver expressões numéricas?

bom vamos entender como resolver uma Expressão numérica na pratica temos a exp:

4 + ( 2 x 2 ) x [ 22 x 2 ] - { 7 + 5 }

nas expressões numéricas são resolvidos primeiro oque esta dentro dos parenteses no nosso caso

4+ ( 2 x 2 ) X [ 22 X 2 ] - { 7 + 5 }

4+4X[22X2]-{7+5}

ok já resolvidos os calculos de dentro dos parenteses resolvemos os colchetes []

vamos lá

4 + 4 x [ 22 x 2 ] - { 7 + 5 }
4 + 4 x 44 - { 7 + 5 }

depois como já deve imaginar resolvemos o que esta dentro das chaves {}


4 + 4 x 44 - { 7 + 5 }
4 + 4 x 44 - 12

depois disso resolvemos as potencias e as raíses no nosso caso não temos

então resolvemos nessa ordem multiplicaçao e divisão depois adição e subtração

vamos lá

4 + 4 x 44 - 12
4 + 176 - 12
180 - 12
= 168

viu como é simples alguns exemplos para praticar :

a ) ( 99 + 99 ) x 2 + 4
b ) 333 + 4 / 2 * [ 22 * 22 ]

lembre sempre da ordem parenteses, colchetes ,chaves , raiz ,potencia,divisão,multiplicação,adição e subtração.

respostas dos exercícios :

a ) ( 99 + 99 ) x 2 + 4
198 x 2 + 4
396 + 4
= 400

b ) 333 + 4 / 2 * [ 22 * 22 ]
333 + 4 / 2 * 484
333 + 2 * 428
333 + 856
= 1189
