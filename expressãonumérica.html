Oque são expressões numéricas?

expressões numéricas são operações matemáticas que sofrem alterações até chegar no resultado final normalmente esse conteúdo caí do 6 ao 7 ano do ensino fundamental e muitos tem dificuldade porque é quando a matemática começa a ficar um pouco mais complexa mais fique tranquilo(a) pois vamos explicar de maneira muito facil de fazer este calculo .

Como calcular teoria

Calculamos sempre primeiro o que esta dentro dos parenteses ( ) , colchetes [ ] e depois o que esta dentro das chaves { } depois e calcular o que estava dentro dos parenteses , colchetes e chaves calculamos as raises e potências depois vamos calcular nessa ordem multiplicação e divisão depois adição e subtração na teoria parece difícil mas na pratica é muito tranquilo .

como calcular pratica :

agora chegou a hora de calcular na pratica

então temos a expressão :

2 x ( 3 + 3 ) + [ 3 x 5 ] - { 4 - 3 }

resolvendo primeiro o que esta dentro dos parênteses :

2 x ( 3 + 3 ) + [ 3 x 5 ] - { 4 - 3 }

2 x 6 + [ 3 x 5 ] - { 4 - 3 }

depois resolvendo oque está dentro dos colchetes

2 x 6 + x [ 3 x 5 ] - { 4 - 3 }

2 x 6 + 15 - { 4 - 3 }

depois resolvemos oque esta dentro das chaves

2 x 6 + 15 - { 4 - 3 }

2 x 6 + 15 - 1

depois resolvemos as potencias e as raises no nosso caso não temos então resolvemos nesta ordem multiplicação,divisão,adição e subtração

2 x 6 + 15 - 1

12 + 15 - 1

= 26
